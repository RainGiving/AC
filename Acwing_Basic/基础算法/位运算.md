# 位运算

## 思路&模板

### 求`n`的二进制表示中的第`k`位（个位视为第`0`位）

1. 先把第`k`位移动到个位：`x = n >> k`
2. 查看个位是几：`x & 1`

```cpp
n >> k & 1
```

### `lowbit(x)`返回一个`x`的第一个1

返回的是一个二进制数，例如`x = 10001000`，`lowbit(x) = 1000`；`x = 11110`，`lowbit(x) = 10`
